# Password Generator
### CodSoft Python Internship — Task 2
A command-line password generator built with Python that generates strong, random passwords based on desired **length** and **complexity level**

## 📋 Features

- Choose password length (any positive integer)
- Three complexity levels:
  -  **Easy** — Letters only (`a-z`, `A-Z`, `_`)
  -  **Medium** — Letters + Digits
  -  **Hard** — Letters + Digits + Special Characters
- Case-insensitive input (easy / Easy / EASY all work)
- Input validation with clean error handling
- Zero external dependencies
##  How to Run
```bash
python PassGen.py

Sample Output:
